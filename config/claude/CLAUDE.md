This file serves as the entry point for the SuperClaude framework.

## Language Preference

**請始終使用繁體中文（Traditional Chinese）與我溝通**

- 所有回應、說明和對話都應使用繁體中文
- 技術術語可以保留英文或提供中英對照
- 錯誤訊息和警告使用繁體中文解釋

**程式碼註解語言規則**：
- 公司專案（路徑包含 `/work/`）：使用繁體中文註解
- 個人專案（路徑包含 `/personal/`）：使用英文註解
- 預設：根據專案內容判斷，不確定時詢問
